Jump Keys

Jump Key Semantics for KEXEC:
  • JK3: If set, the system will NOT attempt an auto-recovery when an IP executes an IAR instruction
  • JK4: On a tape boot KEXEC will reload system libraries from the boot tape
  • JK5: On a recovery boot KEXEC will solicit the name of the device from which to load
  • JK7: KEXEC will re-initialize TIP
  • JK9: Upon boot, KEXEC will clear the various symbiont queues
  • JK13: KEXEC will clear the fixed storage items from the Master File Directory
Jump Keys 1 through 18 are generally reserved for operating system control.
In certain configurations, Jump Keys 19 through 36 are available to applications.
