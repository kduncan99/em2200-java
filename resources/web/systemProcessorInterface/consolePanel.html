Operating System Console

KEXEC Console Commands
Saves all security files to tape, or restores them from tapeDisplays all files queued by the indicated jobDisplays all files queued by the indicated userSets TIP automatic recovery to down, deferred, or upSends a message to a demand session by runid or userid
AP n DNBrings down an application group
AP [ALL | n] FSDisplays the status of one or all application group(s)
AP n UP [REBUILD | SUTIL | INIT]Brings up an application group
AT n CLCloses the audit trail for application group {n}
AT n DFPuts the audit trail for application group {n} into the deferred state
AT n DNPuts a closed audit trail into the down state
AT [ALL | n] FSDisplays the status of one or all application group(s)
AT n INInitializes an audit trail (beginning recovery)
AT n LDDisplays media information for the audit trail
AT [ALL | n] NRPlaces one or all audit trail(s) into deferred state
AT n OPOpens an audit trail
AT n UPChanges an audit trail from down to closed state
B nLimits the maximum number of open batch runs to n
BL [ D ]Displays jobs currently in the backlog
BP 0Clears the system breakpoint
BP l,bdi,offset [ P | R | W ]Sets the system breakpoint and specifies the operation to be trapped
C l,bdi,offset [ value ]Displays or alters the value at a given address
CJ n,...Clears the indicated jump keys
CS A [ runid ]Removes the master hold on backlog, or on a specific job
CS ADRemoves a pevious hold on demand run scheduling
CS ALLRemoves master and all individual holds on jobs in backlog
CS ATRemoves hold on all inactive demand sessions
CS H [ runid ]Sets a master hold on all jobs in backlog, or an individual hold on a specific job
CS HDPlaces a hold on all demand run scheduling
CS HTPlaces a hold on inactive sessions
CS runid*[Px][Dhhmm][Shhmm][Ly]F]Alters the job priority, deadline time, start time, or processor priority for a job in backlog, and/or forces it out of backlog
DDisplays the current system time
D ENTERSpecify at boot time, the system date and time to be used for KEXEC
DJDisplays all of the jump keys currently set
DN node,...Prevents KEXEC from accessing the indicated node(s)
DU [ runid ]Dumps the contents of all allocated memory (or for a specific job) to tape
E jobError terminates a job allowing it to process contingencies
FA deviceRaises a unit attention for the indicated device
FB [ bit ]Displays all or a specific flagbox setting(s)
FB LSTDisplays the list of all flagbox bit names
FB [ ON | OFF ] bitTurns on or off a particular flagbox setting
FF [ dir# ] qualifiter*filename [(fcycle)] [.]Displays information for the given fileset or filecycle
FS nodeDisplays facility status of the given node
FS,[ ADISK | ATAPES | CM | DRS | IOP | IP | MEM | MS | RDISKS | TAPES ]Displays facility status for all nodes of the indicated type
FS,ALL nodeDisplays facility status of the given node and all descendants of that node
HELP commandDisplays detailed usage information for a particular console command
II runid [ instruction ]Interrupts a particular job, optionally sending a particular instruction
IN deviceReinitializes the directory of the pack on the given device
LG [ *runid ] messagePosts a message to the KEXEC log, or to an individual job log
MSDisplays mass storage capacity and usage information
MU [ runid ]Displays memory use for the entire system or for an individual job
MU,ALLDisplays memory use for all individual jobs
RC runidDisplay information pertaining to a specific job
RE runidResume a previously-suspended job
RL banknameReloads a particular commonbank from its system file
RM runidRemove a particular job from backlog
RV nodeSets a particular node to the reserved state
SEC,[ SAVE | RESTORE ]
SJ n,...Sets the indicated jump keys
SM symDisplays the status of the indicated symbiont device
SM sym IInitiate / resume operation of a symbiont device
SM sym LLock out a symbiont device
SP runidSuspends a particular job
SQ [ name ] [ * ]Displays the state of all symbiont devices and queues, or of a particular device or queue
SQ runid [ * ] R
SQ userid [ * ] U
SQ runid [ file ] priorityChanges the priority of all files, or a particular file queued by the indicated job
SQ sname QTO snameRequeues all files queued to the first symbiont device or queue, to the second
SQ runid [ file ] snameRedirects all files or a specific file queued by a job, to the indicated symbiont device or queue
SSDisplay system status
ST[,...] name[,...]Starts a stored runstream with a variety of optional parameters
SU nodeSuspends a node
SX A/F [ name ]Deletes all queued files, or all files on a particular queue
SX A/F runid [ filename ]Deletes all files, or a specific file queued by a particular job
SX A/F userid *UDeletes all files queued by a particular user
T [ D ]Displays status of all batch and demand jobs
T,B [ D ]Displays status of all batch jobs
T,D [ D ]Displays status of all demand jobs
T HDisplays status of all runs held by 'S' option, and by certain other holds
T SDisplays the status of all suspended jobs
TB [ /I ] messageDisplays a message on all active demand terminals
TIP [ DF | DN | UP ]
TIP FSDisplays status of TIP automatic recovery
TIP RSRestarts the TIP application group
TM [ runid/R | userid/U ] message
UP nodeSets the indicated node to the UP state
X {job}Aborts a job, preventing it from processing contingencies
$!Initiates auto-recovery
